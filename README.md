Xbox Game Website (Sage-Inspired WordPress Theme)

This project simulates a production-level WordPress website similar to those used by Xbox Game Studios.

🚀 Features
Custom WordPress theme with Sage-inspired structure
Blade-style templating approach
Custom Post Types (News system)
Modular file organization (app/resources structure)
Front-end styling using SCSS
Build workflow using npm
🔧 Tech Stack
WordPress (PHP)
HTML5 / CSS3 / JavaScript
SCSS
Git / GitHub
📦 Architecture
/app → Theme logic and setup
/resources/views → Template files
/resources/styles → SCSS
/resources/scripts → JS
🔄 Migration Simulation
Exported/imported WordPress content
Reconfigured environment paths
Simulated deployment workflow
🧠 Key Learnings
WordPress theme development using modern structure
PHP templating and dynamic content rendering
Organizing scalable front-end architecture
Integrating development workflows into CMS
